当需要编写 golang 时，通过如下工具获取编写规范和风格指南
```
<use_mcp_tool>
    <server_name>lazygophers</server_name>
    <tool_name>library_get</tool_name>
        <arguments>
        {
          "library_id": "golang/风格指南"
        }
    </arguments>
</use_mcp_tool>
```

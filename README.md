本项目是一个针对 roo code 插件的自定义模块，其中

- `custom_models.yaml` 自定义模式文件，可以通过覆盖的方式覆盖当前本地的全局/项目配置使用
- `roles` 是自定义的角色，使得恢复的内容更加的人性化

## 模式说明

- [custom_models](custom_models) 用于存放每一个自定义模式，每个模式是一个 yaml 文件，格式如下:

| 模式名称          | 角色定义                | 使用场景                        | 关键功能                                                                 | 文件路径                  | slug         |

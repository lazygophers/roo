# 模式选择

> **专业分工，智能协作**：Roo Code 系统提供 14 个专业化模式，覆盖从系统架构到代码实现的完整开发流程

## 模式分类

| 类别           | 模式数量 | 核心职责                       |
| -------------- | -------- | ------------------------------ |
| **通用模式**   | 9        | 任务调度、架构设计、研究分析等 |
| **编程子模式** | 5        | 特定编程语言的代码实现         |

---

## 🧠 核心模式详解

### 1. 🧠 Brain (orchestrator) - 智能总指挥

**核心定位**：系统级任务调度中枢，专注于复杂任务的智能分解与模式协作管理。

**核心能力**：

- **任务分解**：将复杂任务拆解为可执行的子任务
- **模式协调**：根据任务性质匹配合适的专业模式
- **进度追踪**：实时监控任务状态和执行进度
- **结果整合**：汇总各子任务结果，形成最终交付

**适用场景**：

- 复杂项目的整体规划和任务分解
- 跨模式协作的调度管理
- 需要多专业领域协作的综合任务

### 2. 🏗️ Architect (architect) - 顶尖架构师

**核心定位**：系统架构设计专家，专注于技术选型、架构设计和系统规划。

**核心能力**：

- **架构设计**：系统整体架构和技术方案设计
- **技术选型**：根据需求选择最适合的技术栈
- **UML 建模**：创建标准化的系统设计文档
- **ADR 管理**：架构决策记录和管理

**适用场景**：

- 新系统的架构设计和技术选型
- 现有系统的重构和优化
- 复杂业务场景的架构方案设计

### 3. 📚 Academic Advisor (ask) - 学术顾问

**核心定位**：知识解释和概念探索专家，提供教科书式的详细解答。

**核心能力**：

- **深度解释**：提供详细、准确的技术概念解释
- **可视化表达**：使用图表和示例增强理解
- **多源知识**：整合多个知识源提供全面解答
- **教学导向**：以教学方式传递复杂概念

**适用场景**：

- 复杂技术概念的学习和理解
- 代码实现原理的深入分析
- 技术方案的理论基础研究

### 4. 🔬 Exception Analyst (debug) - 异常分析师

**核心定位**：问题诊断和调试专家，专注于系统化的问题排查和解决。

**核心能力**：

- **系统化分析**：采用科学方法进行问题诊断
- **假设驱动**：基于假设的验证式调试流程
- **根因分析**：深入挖掘问题的根本原因
- **预防措施**：提供问题预防的最佳实践

**适用场景**：

- 复杂的 Bug 排查和修复
- 系统性能问题的诊断
- 代码质量和安全问题的分析

### 5. 📝 Documentation Engineer (doc-writer) - 文档工程师

**核心定位**：技术文档专家，专注于高质量文档的创建和管理。

**核心能力**：

- **结构化写作**：创建逻辑清晰的文档结构
- **标准化格式**：遵循行业标准的文档格式
- **质量管理**：确保文档的准确性和完整性
- **多格式输出**：支持多种文档格式的生成

**适用场景**：

- API 文档的编写和维护
- 技术规范和用户手册的创建
- 项目文档的整理和优化

### 6. 🔍 Project Researcher (project-research) - 项目研究员

**核心定位**：代码库分析专家，专注于现有系统的深入理解和分析。

**核心能力**：

- **代码结构分析**：深入理解代码库的组织结构
- **依赖关系梳理**：分析模块间的依赖关系
- **技术栈识别**：识别项目使用的技术栈和工具
- **重构建议**：基于分析结果提供改进建议

**适用场景**：

- 接手新项目的快速了解
- 代码库的深度分析和评估
- 技术债务的识别和管理

### 7. 🧪 Chief Researcher (researcher) - 首席研究员

**核心定位**：系统性研究专家，专注于技术方案的深入研究和分析。

**核心能力**：

- **多维分析**：从多个维度分析技术方案
- **竞品分析**：对比分析不同解决方案的优劣
- **趋势研究**：把握技术发展趋势和最佳实践
- **决策支持**：为技术决策提供全面的研究支持

**适用场景**：

- 重大技术选型的决策支持
- 新技术的研究和评估
- 行业最佳实践的总结和应用

### 8. 🛠️ Mode Engineering Master (mode-writer) - 模式工程大师

**核心定位**：模式创建专家，专注于自定义模式的开发和维护。

**核心能力**：

- **模式设计**：设计新的专业化模式
- **提示词工程**：优化模式的提示词配置
- **测试验证**：确保新模式的质量和效果
- **迭代优化**：基于反馈持续改进模式

**适用场景**：

- 创建针对特定领域的新模式
- 优化现有模式的性能和效果
- 模式系统的维护和升级

### 9. 🔄 Version Control Expert (giter) - 版本控制专家

**核心定位**：Git 工作流专家，专注于版本控制和协作开发的管理。

**核心能力**：

- **Git 工作流**：精通各种 Git 工作流和最佳实践
- **分支管理**：合理的分支策略和管理
- **冲突解决**：高效处理代码冲突
- **协作规范**：团队协作的标准化流程

**适用场景**：

- 大型项目的版本控制管理
- 团队协作开发的流程规范
- 复杂的合并和冲突处理

## 💻 编程子模式详解

### 10. 🐍 Python Code Magician (code-python) - Python 代码魔法师

**核心定位**：Python 开发专家，专注于 Python 生态系统的深度应用。

**核心能力**：

- **Python 生态**：精通 Python 及其主流框架
- **设计模式**：熟练应用 Python 设计模式
- **性能优化**：Python 代码的性能调优
- **工程实践**：Python 项目的工程化最佳实践

**技术栈**：

- **Web 框架**：FastAPI, Django, Flask
- **数据处理**：Pandas, NumPy, SciPy
- **异步编程**：asyncio, aiohttp
- **测试框架**：pytest, unittest

**适用场景**：

- Python 后端 API 开发
- 数据分析和处理脚本
- 自动化工具和脚本开发
- Python 包的开发和维护

### 11. ⚛️ React Code Magician (code-react) - React 代码魔法师

**核心定位**：React 开发专家，专注于现代 React 应用的开发。

**核心能力**：

- **React 生态**：深度理解 React 生态系统
- **TypeScript 集成**：React + TypeScript 的最佳实践
- **现代开发**：Hooks, Context, 状态管理
- **性能优化**：React 应用的性能优化策略

**技术栈**：

- **核心框架**：React 18+, Next.js
- **状态管理**：Redux, Zustand, Jotai
- **样式方案**：Styled Components, Tailwind CSS
- **构建工具**：Vite, Webpack

**适用场景**：

- React 单页应用开发
- Next.js 全栈应用开发
- React 组件库开发
- React Native 移动应用开发

### 12. ☕ Java Code Magician (code-java) - Java 代码魔法师

**核心定位**：Java 开发专家，专注于企业级 Java 应用的开发。

**核心能力**：

- **JVM 生态**：深度理解 Java 虚拟机和生态系统
- **企业级开发**：Spring 生态和企业级应用开发
- **并发编程**：Java 并发编程和多线程处理
- **性能调优**：JVM 性能调优和内存管理

**技术栈**：

- **核心框架**：Spring Boot, Spring Cloud
- **持久层**：JPA, Hibernate, MyBatis
- **构建工具**：Maven, Gradle
- **测试框架**：JUnit, TestNG

**适用场景**：

- 企业级后端服务开发
- 微服务架构应用开发
- 大型分布式系统开发
- Java 库和框架开发

### 13. 🦀 Rust Code Magician (code-rust) - Rust 代码魔法师

**核心定位**：Rust 开发专家，专注于系统级和高性能应用开发。

**核心能力**：

- **内存安全**：Rust 内存安全保证和所有权系统
- **并发性能**：Rust 并发编程和高性能处理
- **系统编程**：底层系统编程和接口开发
- **零成本抽象**：Rust 零成本抽象和泛型编程

**技术栈**：

- **Web 框架**：Axum, Actix Web
- **异步运行时**：Tokio, async-std
- **系统库**：nix, libc, winapi
- **工具链**：Cargo, Rust Analyzer

**适用场景**：

- 系统级工具和库开发
- 高性能 Web 服务开发
- 区块链和加密应用开发
- 嵌入式系统开发

### 14. 💚 Vue Code Magician (code-vue) - Vue 代码魔法师

**核心定位**：Vue 开发专家，专注于 Vue 生态系统的应用开发。

**核心能力**：

- **Vue 生态**：深度理解 Vue 3 生态系统
- **组件开发**：Vue 组件设计和开发
- **状态管理**：Pinia 状态管理和数据流
- **工程化**：Vue 项目的工程化配置

**技术栈**：

- **核心框架**：Vue 3, Nuxt.js
- **构建工具**：Vite, Vue CLI
- **UI 框架**：Element Plus, Ant Design Vue
- **状态管理**：Pinia, Vuex

**适用场景**：

- Vue 单页应用开发
- Nuxt.js 全栈应用开发
- Vue 组件库开发
- Vue 插件和工具开发

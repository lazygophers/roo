# Git Commit Command

根据用户所描述的 `<scope>` 提交代码变更。如果用户没有描述范围，则默认只提交之前的步骤中产生变更的文件。

提交行为应交由 `giter` 模式执行。

本指令遵循 **Conventional Commits** 规范，提供标准化的 Git 提交工作流，并通过 AI 增强功能，实现高效、智能、规范的提交体验。

## AI 增强功能

本工具集成了 AI 能力，可以智能分析代码变更并生成符合规范的提交信息。
